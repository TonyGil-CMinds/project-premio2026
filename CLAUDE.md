## Architecture

Single-page Next.js 14 hero landing page (app router) for Premio Natura500 — a regenerative innovation award for Latin America/Caribbean. The entire UI lives in one hero section with no additional routes.

**File layout:**
- `app/layout.tsx` — Root layout, metadata, Google Fonts (Host Grotesk + Unbounded), Spanish lang
- `app/page.tsx` — Home page, renders `<PremioHero />`
- `app/globals.css` — All styles, CSS custom properties, responsive breakpoints
- `components/PremioHero.tsx` — Main orchestrator: layout, nav, CTA, entrance animation timeline
- `components/PremioLogo.tsx` — Animated SVG logo (GSAP staggered entrance)
- `components/CountdownGlass.tsx` — Live countdown to July 25, 2026 with pointer-tracking parallax

## Key Patterns

**Animation stack:** GSAP is the primary animation library. All entrance animations use a single GSAP timeline in `PremioHero.tsx` (`gsap.context()` with cleanup on unmount). Interactive animations (pointer tracking in `CountdownGlass`) use `gsap.to()` directly.

**Liquid Glass:** The `liquid-glass-react` package wraps the nav mark, nav links pill, and countdown. Each instance has tuned `displacementScale`, `blurAmount`, `saturation`, and `elasticity` values — treat these as design-sensitive; avoid normalizing them.

**CSS variables** (defined in `:root` in `globals.css`):
- `--green-bright: #a4d73a` — primary accent
- `--yellow: #ffdd00` — CTA button
- `--font-host` / `--font-unbounded` — font family variables from Next.js font loader

**Responsive breakpoints:** `max-width: 920px` (tablet) and `max-width: 560px` (mobile). Some nav links are hidden at smaller breakpoints via `display: none`.

**Countdown target:** Hardcoded in `CountdownGlass.tsx` as `2026-07-25T06:00:00-06:00`. Update this when the deadline changes.
